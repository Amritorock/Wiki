### Clean Flash
```
1. Reboot to the new recovery (if it didn't reboot itself).
2. Wipe system, vendor, metdata, cache.
3. Flash rom
4. Format data
5. Reboot
6. Enjoy :)
```

### Dirty Flashing Steps

```
1. Flash the rom
2. Wipe dalvik, cache
3. Reboot
4. Enjoy
```

### Note:
```
- If you are coming from A12/A13 to A14 then clean flash is compulsory and format data.
```
